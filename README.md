## Dev environment

* ruby 3.2.2 (2023-03-30 revision e51014f9c0) [x86_64-darwin20]
* Xcode: Version 12.5.1 (12E507)


```
rvm install "ruby-3.2.2" --with-openssl-dir="$(brew --prefix)/opt/openssl@1.1/"
```

Install dependency

```
pod install
```
